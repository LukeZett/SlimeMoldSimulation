# SlimeMoldSimulation
Slime mold simulation in C++ and WebGPU.

## Features
- Simple UI to control simulation parameters
- Compute shaders to handle milions agents simultaniously (3 200 000 used by default)
- WebGPU graphic API ensuring portability
